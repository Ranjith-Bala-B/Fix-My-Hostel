# Hostel Complaint Management System

A Flask web application for managing hostel complaints for SECE students.

---

## 🚀 Features

- Student login with official SECE emails
- Submit complaints with category, priority, description, and image upload
- Admin dashboard with charts (category, priority, monthly trends)
- Admin can update status, add notes, or delete complaints
- PostgreSQL database via Render

---

## 🚀 Deployment (Render)

1. Push this repo to GitHub  
2. Go to https://render.com → New → Blueprint  
3. Connect your GitHub repo — Render will auto-detect `render.yaml`  
4. It will create the PostgreSQL database and web service automatically  

---

## ⚙️ Environment Variables (auto-set by render.yaml)

| Variable        | Description                                  |
|----------------|----------------------------------------------|
| DATABASE_URL   | PostgreSQL connection string (auto-injected) |
| SECRET_KEY     | Auto-generated by Render                     |
| ADMIN_PASSWORD | Default: `Admin@1234`                        |
| FLASK_ENV      | `production`                                 |

---

## 💻 Local Development

```bash
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
App runs at: http://localhost:5000
